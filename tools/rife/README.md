# RIFE tool

Place `rife-ncnn-vulkan.exe` and its required DLL files here.

RIFE model files do not belong here. Runtime model paths must point to:

`assets/models/video_interpolation/rife/`

Search order:

1. User configured RIFE executable path.
2. `tools/rife/rife-ncnn-vulkan.exe`.

The app does not run RIFE from the external reference asset library.
