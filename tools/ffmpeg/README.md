# FFmpeg tools

Put `ffmpeg.exe` and `ffprobe.exe` here if you want HEA to use project-local media tools.

Search order:

1. Paths configured by the user in settings.
2. `tools/ffmpeg/ffmpeg.exe` and `tools/ffmpeg/ffprobe.exe`.
3. System `PATH`.

The executable files are not committed to Git. They are runtime tools, not model files.
